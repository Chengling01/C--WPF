# 14.1可视化对象
## 14.1.1绘制可视化对象
Visual类是抽象类，为了使用DrawingVisual类绘制内容，需要调用DrawingVisual.RenderOpen方法。该方法返回一个用于定义可视化内容的DrawingContext对象。绘制完毕后需要调用Close方法。绘制完毕后不可修改绘制的内容，但是可以改变绘制内容的Transform和Opacity属性。
以下将列举DrawingContext类的方法：

|                                    名称                                    |                                                 说明                                                 |
| :----------------------------------------------------------------------: | :------------------------------------------------------------------------------------------------: |
| DrawLine()<br>DrawRectangle()<br>DrawRoundedRectangle()<br>DrawEllipse() |                                      在指定的位置，使用指定的填充和轮廓绘制特定的形状                                      |
|                     DrawFeometry()<br>DrawDrawing()                      |                                     回执更复杂的Geometry对象和Drawing对象                                     |
|                                DrawText()                                | 在指定的位置绘制文本。通过该方法传递FormattedText对象，可指定文本、字体、填充以及其他细节。如果设置了FormattedText.MaxTetWidth属性，可使用该方法绘制换行的文本 |
|                               DrawImage()                                |                                           在指定区域绘制一副位图图像                                            |
|                               DrawVideo()                                |                                            在特定区域绘制视频内容                                             |
|                                  Pop()                                   |                    翻转最后调用的PushXxx（）方法。可使用PushXxx方法暂时应用一个或者多个效果，并使用Pop()方法翻转它们。                     |
|                                PushClip()                                |                                    将绘图限制在特定剪切区域中。这个区域外的内容不会被绘制                                     |
|                               PushEffect()                               |                                      为随后的绘图操作应用BitmapEffect对象                                      |
|                    PushOpacity()<br>PishOpacityMask()                    |                                 为了使用后续的绘图将操作部分透明，应用新的不透明设置或者不透明的掩码                                 |
|                             PushTransform()                              |                            设置将应用于后续绘制操作的Transform对象。可使用变换来缩放、移动、旋转、扭曲等                             |
创建可视化对象示例：
```Csharp

//绘制图形时需要在visual.RenderOpen();与dc.Close();之间进行
 private void Button_Click_3(object sender, RoutedEventArgs e)
 {
     DrawingVisual visual = new DrawingVisual();
     DrawingContext dc = visual.RenderOpen();
     Pen drawingPen = new Pen(Brushes.Black, 3);
     dc.DrawLine(drawingPen,new Point(0,50),new Point(50,0));
     dc.DrawLine(drawingPen, new Point(50, 0), new Point(100, 50));
     dc.DrawLine(drawingPen, new Point(0, 50), new Point(100, 50));
     dc.Close();
     cas.AddVisual(visual);
 }

```

==注意：绘制好的图形无法直接添加到UI界面，需要在Canvas的基础上派生出存放可视化对象的面板==
```Csharp

 public class DrawingCanvas:Canvas
 {
     private List<Visual> visuals=new List<Visual>();

     protected override int VisualChildrenCount
     {
         get { return visuals.Count; }
     }

     protected override Visual GetVisualChild(int index)
     {
         return visuals[index];
     }

     public void AddVisual(Visual visual)
     {
         visuals.Add(visual);
         base.AddVisualChild(visual);
         base.AddLogicalChild(visual);
     }

     public void RemoveVisual(Visual visual)
     {
         visuals.Remove(visual);
         base.RemoveLogicalChild(visual);
         base.RemoveLogicalChild(visual);
     }
 }

```
```XAML

<Grid>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="120"/>
        <ColumnDefinition Width="*"/>
    </Grid.ColumnDefinitions>
    
    <StackPanel>
        <Button Content="绘制图形" Height="30" Width="120" Click="Button_Click_3"/>
    </StackPanel>

    <local:DrawingCanvas x:Name="cas" Background="White" ClipToBounds="True" Grid.Column="1"></local:DrawingCanvas>
    
</Grid>
```

## 14.1.2命中测试
为支持命中测试，需要为DrawingCanvas类添加GetVisual（）方法。该方法是使用一个点作为参数并返回匹配的DrawingVisual对象。为此使用VisualThreeHelper.HitTest()静态方法。
```Csharp
//传入的点需要是相对于存放面板的像素点
Pubulic DrawingVisual GetVisual(Point point)
{
  HitTestResult hitResult=VisualTreeHelper.HitTest(this,point);
  return hitResult.VisualHit as DrawingVisual;
}
```

# 14.2效果
WPF提供了可应用于任何元素的可视化效果。效果的目标是提供一种简便的声明式方法，从而改进文本、图像、按钮以及其他控件的外观。
下表列举出可使用的效果类：

|        名称        | 说明                                     | 属性                                                               |
| :--------------: | :------------------------------------- | ---------------------------------------------------------------- |
|    BlurEffect    | 模糊元素中的内容                               | Radius、KernelType、RenderingBias                                  |
| DropShadowEffect | 在元素背后添加矩形阴影                            | BlurRadius、Color、Direction、Opacity、<br>ShadowDepth、renderingBias |
|   ShaderEffect   | 应用像素着色器，像素着色器是使用该机着色语言实现制作好的并且已经编译过的效果 | PixelShader                                                      |
## 14.2.1BlurEffect类
该类模糊元素的内容，就像通过失焦透镜观察到的效果。通过Radius属性的值可增加模糊的程度。示例：
```XAML

 <Button Content="Blurred" Padding="5" Margin="3">
     <Button.Effect>
         <BlurEffect Radius="5" KernelType="Gaussian" RenderingBias="Quality"/>
     </Button.Effect>
 </Button>

```

### 14.2.2DropShadowEffect类
DropShadowEffect类在元素背后添加了轻微的偏移阴影。可使用该类的以下几个属性。

|      名称      | 说明                                                                                   |
| :----------: | :----------------------------------------------------------------------------------- |
|    Color     | 设置阴影的颜色，默认为黑色                                                                        |
| ShadowRadius | 确定阴影离开内容多远，单位为像素（默认值为5）。将该属性设置为0会创建外侧辉光的效果，该效果会在内容周围添加晕彩。                            |
|  BlurRadius  | 模糊阴影，该属性和BlurEffect类的Radius属性非常类似                                                    |
|   Opacity    | 使用从1到0之间的小数，使阴影部分透明                                                                  |
|  Direction   | 使用从0到360之间的角度值指定阴影相对于内容的位置。该属性设置为0会将阴影放置到右边，增加该属性的值时会逆时针移动阴影。默认值时315，该值会将阴影放置到元素的右下方 |

示例：
```XAML
 <StackPanel>
     <TextBlock FontSize="20" Margin="5">
         <TextBlock.Effect>
             <DropShadowEffect></DropShadowEffect>
         </TextBlock.Effect>
         <TextBlock.Text>Basic dropshasdow</TextBlock.Text>
     </TextBlock>

     <TextBlock FontSize="20" Margin="5">
         <TextBlock.Effect>
             <DropShadowEffect Color="SlateBlue"/>
         </TextBlock.Effect>
         <TextBlock.Text>Ligth blue dropshadow</TextBlock.Text>
     </TextBlock>

     <TextBlock FontSize="20" Margin="5" Foreground="White">
         <TextBlock.Effect>
             <DropShadowEffect BlurRadius="15"></DropShadowEffect>
         </TextBlock.Effect>
         <TextBlock.Text>Blurred dropshadow with white text</TextBlock.Text>
     </TextBlock>

     <TextBlock FontSize="20" Foreground="Magenta" Margin="5">
         <TextBlock.Effect>
             <DropShadowEffect ShadowDepth="0"/>
         </TextBlock.Effect>
         <TextBlock.Text>Close dropshadow</TextBlock.Text>
     </TextBlock>

     <TextBlock FontSize="20" Foreground="LimeGreen" Margin="5">
         <TextBlock.Effect>
             <DropShadowEffect ShadowDepth="25"></DropShadowEffect>
         </TextBlock.Effect>
         <TextBlock.Text>Distant dropshadow</TextBlock.Text>
     </TextBlock>
 </StackPanel>
```
